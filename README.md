# 方法

週に一回2人づつ蟻本から好きなテーマを選び発表する。  
発表するテーマは2人で関連するものでも良いし、そうでなくても良い。  
スクリーンと(黒/白)板は利用できる。  
発表時間は1人30分以下とし、残りの時間は演習の時間とする。  
よって発表では演習問題の提示があると良い。  
宿題があっても良い。  

質問はコメント欄か @siquare が受け付ける。
